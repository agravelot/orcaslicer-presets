# orcaslicer-presets

Generate custom OrcaSlicer presets for machine and process profiles.

## Project layout

- `cmd/main`: generates user profiles into `user/default`
- `machine`: machine profile model + generation logic
- `process`: process profile model + generation logic
- `filament`: filament profile model (generation not wired yet)
- `internal/output`: shared JSON/info profile writer
- `utils`: helper utilities and tests

## Usage

Run profile generation:

```bash
go run ./cmd/main
```

This writes generated files under:

- `user/default/machine/*.json` + `.info`
- `user/default/process/*.json` + `.info`

Currently, `filament` output files are not generated by `cmd/main`.

## Development

Run tests:

```bash
go test ./...
```

## Notes

- Source-compatible with Orca profile keys via typed/optional struct fields.
- Refactor plan and progress are tracked in `REFACTOR_PLAN.md`.
- Existing generated files under `user/default` may include local machine-specific values (for example API keys); review before sharing.
